How to find things...(Hint: use 'Ctrl + F')

The most common request for any prjectdocumentation is to be able to find things. This portal is 'all in one page', which means if it is not in this page, it does not exist. This means you can use the regular 'Find...' tool already built into your browser to find text.

For Internet Exploter, Google Chrome and FireFox type Ctrl + F to open the search tool, and enter any key words you wish to find.

You can also use the menu in the sidebar or the menu in the navbar at the top. They are identical but the sidebar will disappear if you are a mobile phone or make your browser very narrow.
